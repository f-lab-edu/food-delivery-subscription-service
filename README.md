# 음식 배달 구독 서비스

음식 배달 구독 서비스 입니다. 근처 음식점의 음식 메뉴를 선택해서 원하는 요일과 시간을 사용자가 지정한 후, 사용자 구독 정보에 따라서 음식을 배달합니다.

## 사용 기술 및 개발 환경

- Java 17
- Spring Boot 2.6.7
- IntelliJ
- gradle
- MyBatis
- MySQL
