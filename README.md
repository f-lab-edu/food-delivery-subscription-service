# FOOD-RUN

음식 배달 서비스 입니다. 음식점의 메뉴를 선택해 집 앞으로 배달하는 서비스입니다.

## 사용 기술 및 개발 환경

- Java 17
- Spring Boot 2.6.7
- IntelliJ
- gradle
- MyBatis
- MySQL
